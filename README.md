# Investment Portfolio Analysis Dashboard (Excel)

## Project Overview

This project showcases an interactive **Investment Portfolio Analysis Dashboard** built in **Microsoft Excel**. The dashboard transforms raw investment data into meaningful insights using Pivot Tables, Pivot Charts, slicers, formulas, and KPI cards.

The objective is to help stakeholders monitor investment performance, identify trends, compare investment categories, and support data-driven financial decisions.


## Business Objectives

This dashboard answers important business questions such as:

- What is the total investment amount?
- How much interest has been accrued?
- Which investment category performs best?
- Which city has the highest investment value?
- How have investments changed over time?
- How many investors are represented in the dataset?

# Location Analysis

The dashboard compares investments across multiple cities including:

- Lagos
- Ibadan
- Abuja
- Port Harcourt
- Enugu
- Onitsha

### Key Insight

Lagos recorded one of the highest investment totals, with strong contributions from the other major cities. Investment activity is well distributed across the locations represented in the dataset.


# Investment Type Analysis

Investment categories include:

- Stocks
- Bonds
- Mutual Funds
- Crypto
- Real Estate

### Key Insight

Mutual Funds recorded the highest total investment amount, followed by Crypto and Bonds, indicating strong investor preference for diversified investment products.


# Yearly Investment Trend

The dashboard tracks yearly investment performance from **2018–2024**.

### Key Insight

Investment activity remained strong over the observed period, providing a useful basis for comparing yearly investment levels and accrued interest.


# Business Insights

- Over **₦2.54 Billion** was invested across the dataset.
- Accrued interest exceeded **₦1.17 Billion**.
- The dashboard analyzes investment activity for **1,000 investors**.
- Mutual Funds attracted the highest investment volume.
- Investment performance can be analyzed by year, investment type, and location using interactive filters.
